Ans by Abrahm Williams

Everyday calculator for everyone

Ans is a simple calculator app that handle tasks well, is suitable for everyday use and can be downloaded free without cost. It is developed with web technologies and available as a webapp for various devices − firefox os, android etc on the Firefox Marketplace.

You can use the default calculator for any computation and force precedence using parenthesis. Most features you want to know are

 
Error

Expresses different kinds of errors

Options

Swipe up when needed to do some advanced computation

Operator

Evaluates or breaks down multiple operators

Result

Outputs non-precision JavaScript result

And much more...

Development

Ans is my personal contribution to Firefox Marketplace, first released in 2015 as Ansii and continiously developed with few resources, of course many thanks goes to Github repositories Entities, Fira, developement tools used − Notepad++, Firefox and Firefox OS simulator.

Ans app instills notepad look and feel as in recent releases and only returned faster in this version with few UI issues, no external framework is used in this development both recent and latest apps use JavaScript (vanilla) binaries

This work is dedicated to Family, Institutions, to God be all the glory.

Download Ans version 1.5 from Firefox Marketplace or get the packaged distribution, Ans may require that you have a web runtime such as Firefox Browser installed to start the application
