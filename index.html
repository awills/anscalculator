Ans 1.5  by Abrahm Williams

Everyday calculator for everyone

Ans is a simple calculator app that handle competitive tasks well, is suitable for everyday use and free. It is developed with web technologies and available for download as a webapp for various devices − firefox os, android on the firefox marketplace.

Most features you want to know

Precedence

Whether its casual or professional use, you should get the result that is right with recognition of operators and option to force such precedence using parenthesis

Error

You will find out different kind of error notice appear

Math operations

When needed, do some advance computation

Cropping

Crop back to some point in the expression when critical

Increment

Continue pressing equal-to when you don't want to retype

Developement

My many thanks goes to github repositories Entities, Fira. Developement tools used − Notepad++, Firefox and Firefox os simulator.
